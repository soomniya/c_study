### 어셈블리어
- [ ] 재밌긴 한디... 다시 봐보자....
* 00CE1043 cmp: 빼기라고 보면 됨, 왼쪽에서 오른쪽을 뺀 값이 0과 같은지 확인
* 00CE1047 je: jz ==> jump if zero 00CE105B
* 00CE105B xor: 피연산자가 같으면 xor 결과 == 0
  * eax: 반환값을 저장하는 레지스터
* 00CE105D pop: ebp
* 00CE105E ret: return
